🔥 Next.js + TypeScript + Firebase 기반 포켓몬 카드 웹 서비스

이 프로젝트는 Next.js(Typescript) 와 Firebase 를 활용하여 포켓몬 카드 데이터를 불러오고,
간단한 회원가입/로그인 기능과 즐겨찾기(Favorite) 기능을 제공하는 웹 서비스입니다.

✅ 주요 기능

포켓몬 카드 리스트 불러오기

외부 API를 통해 포켓몬 카드 데이터를 가져와 카드 형태로 표시합니다.

Firebase 회원 인증

이메일/비밀번호 기반의 간단한 회원가입 및 로그인 기능 제공

즐겨찾기 기능

로그인한 사용자는 원하는 포켓몬 카드를 즐겨찾기에 추가/삭제 가능

즐겨찾기 목록은 Firebase Firestore에 저장되어 사용자별로 관리됩니다.

반응형 UI

다양한 화면 크기에서 사용 가능한 직관적이고 깔끔한 인터페이스 제공

✅ 기술 스택

Next.js (App Router)

TypeScript

Firebase Authentication

Firebase Firestore

CSS / Tailwind / Chakra UI 등 (사용 중인 UI 라이브러리)
